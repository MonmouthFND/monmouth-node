# `monmouth-transport-sim`

[![CI](https://github.com/monmouth-ai/monmouth/actions/workflows/ci.yml/badge.svg)](https://github.com/monmouth-ai/monmouth/actions/workflows/ci.yml)
[![License: MIT](https://img.shields.io/badge/License-MIT-blue.svg)](https://opensource.org/licenses/MIT)

Simulated P2P transport for Monmouth node testing and development.

## Key Types

- `SimMarshalChannels` - Marshaled channel abstraction for simulated transport
- `SimSimplexChannels` - Simplex channel implementation for one-way communication
- `SimContext` - Context for managing simulated transport state
- `SimTransportError` - Error types for simulated transport operations
- `SimTransportProvider` - Provider implementation for simulated transport
- `SimControl` - Control interface for managing simulated network behavior
