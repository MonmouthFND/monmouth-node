# `kora-runner`

[![CI](https://github.com/refcell/kora/actions/workflows/ci.yml/badge.svg)](https://github.com/refcell/kora/actions/workflows/ci.yml)
[![License: MIT](https://img.shields.io/badge/License-MIT-blue.svg)](https://opensource.org/licenses/MIT)

Production node runner for Kora validators.

## Key Types

- `RevmApplication` - REVM-based application execution environment
- `RunnerError` - Error types for runner operations
- `ProductionRunner` - Main production node runner implementation
- `ThresholdScheme` - Threshold signature scheme configuration
